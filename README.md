# WPL Session and Cookies

# 🔐 User Authentication System (PHP + MySQL)

A secure and efficient web application that implements **user authentication** using PHP and MySQL.  
This project demonstrates **session management, cookie handling, and security practices** such as password hashing and input validation.

---

## 📌 Features

- ✅ User Registration (with password hashing)
- ✅ User Login Authentication
- ✅ Session Management (PHP Sessions)
- ✅ "Remember Me" functionality using Cookies
- ✅ Input Validation (JavaScript + PHP)
- ✅ Protection against SQL Injection (Prepared Statements)
- ✅ Basic XSS Prevention (`htmlspecialchars`)
- ✅ Responsive and Modern UI

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** XAMPP / WAMP / MAMP
